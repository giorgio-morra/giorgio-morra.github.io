Project summary

This project builds a fully automated computer vision workflow for license plate detection and character extraction. The pipeline performs image preprocessing, rectangular contour selection, perspective warp, and character segmentation without manual interaction.

First, the image is converted to grayscale, equalized, and Gaussian blurred to enhance contrast and reduce noise. Canny edge detection and morphological filtering isolate the plate boundary, while contour approximation selects the best rectangular plate candidate.

The selected plate region is then corrected with a perspective transform and cropped to the license plate area. Character zones are binarized and segmented using contour filtering, then compared to a reference database via pixel-wise template matching.

Key features

  • Gaussian blur and histogram equalization for noise reduction and contrast enhancement
  • Canny edge detection with morphological closure to isolate the plate contour
  • Contour approximation to select the best rectangular license plate candidate
  • Perspective transform to rectify and crop the plate region
  • Binarization and contour filtering for reliable character segmentation
  • Template matching against a letter/number database using pixel-wise similarity
