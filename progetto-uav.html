Project Summary

This project focuses on the development of a complete closed-loop 3D navigation and control architecture for autonomous vehicles. The pipeline is designed to safely navigate complex environments by bridging the gap between global path planning and real-time kinodynamic trajectory execution.

The work began by constructing a 3D occupancy grid to map static environments, utilizing an A* search algorithm to generate a collision-free coarse route. To ensure dynamic feasibility, a visibility-based shortcutting algorithm and cubic-spline interpolation were applied, transforming the raw grid path into a smooth, time-parameterized reference trajectory containing strict position and velocity targets.

Building upon this global planning framework, a receding-horizon Model Predictive Controller (MPC) was developed using CasADi and IPOPT to handle real-time trajectory tracking. The controller actively optimizes acceleration commands while enforcing strict kinematic bounds and hard constraints for collision avoidance. Crucially, the MPC is capable of predicting and reacting to unmapped, moving obstacles in real-time, ensuring safe autonomous operation even in highly dynamic scenarios.

Key Contributions

  • Autonomy Architecture: Engineered a navigation pipeline from 3D global routing to real-time trajectory optimization for autonomous vehicles.
  • Path Planning and Trajectory Optimization: Implemented an A* global path planner with shortcutting and cubic-spline smoothing to generate time-parameterized reference trajectories.
  • Non-Linear MPC Optimization: Developed a receding-horizon controller utilizing CasADi/IPOPT to execute precise trajectory tracking under strict speed and acceleration constraints.
  • Dynamic Obstacle Avoidance: Integrated reactive collision avoidance constraints within the MPC, enabling the vehicle to safely bypass moving targets unknown to the global planner.
